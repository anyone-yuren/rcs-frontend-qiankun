### **Use**

想要在你的应用中使用组件，你需要动态的使用`import`从 components 包中导入你想要的组件：

`import { MxwCar } from 'components';`

and use them like so
然后像这样使用

```
  const example = () => (
    <div>
      <MxwCar />
    </div>
  )
```

当然不要忽视了可能给你提示的报错或警告（TS）
